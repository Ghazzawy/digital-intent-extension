**Digital Intent Companion** is a lightweight browser extension designed to help users stay mindful and renew their intention while browsing the web.  
The goal is to encourage healthy digital habits and conscious use of time online through gentle reminders and positive prompts.

This project is also a learning-focused initiative to practice real-world software development, browser extension architecture, and responsible AI usage.

---

## Current Status
**Early MVP (Minimum Viable Product)**

---

## Core Idea
While a user is browsing:

- Detect the type of website (education, social media, entertainment, etc.)
- After a period of time, show a small reminder popup
- Encourage intentional and mindful usage of time
- Everything runs locally on the device with **no data collection**
